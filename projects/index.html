My projects

Kotlin events

kotlinx.events is a small, simple event library for Kotlin, focused on ease of use and familiarity. It provides a high-level façade to abstract away the details of handling events and event listeners, analogously to the event language feature in C#.

memistry

A web-based chemistry trainer, written mostly in a day for self-study purposes.

memistry provides randomized questions to aid learning or memorization; currently, it generates questions concerning the periodic table, its elements, and their properties.

GDPR

A summary of the General Data Protection Regulation (Regulation (EU) 2016/679 of the European Parliament and of the Council of 27 April 2016). Condenses the 90 pages of the GDPR into about 40 pages, with fully linked references and less legalese to sift through.

Melange

A documentation generator for the Clang frontend. Written using LLVM 9/10 LibTooling and generates HTML5 documentation using Mustache.

This project has not been finished and is not currently public.

ringcode

A circular, color-based tag format for encoding bytes. Consists of a generator and a reader written in JavaScript with the Canvas API and OpenCV's Emscripten bindings. This was conceived and written as part of a school project involving augmented reality and Microsoft's HoloLens™.

While it is not particularly efficient with respect to storage density, it functioned primarily as a learning experience, and as a proof-of-concept for the cross-platform usage of OpenCV.
